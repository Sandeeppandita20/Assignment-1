//Readme for command gpt chat 

Node.js and npm installed should be installed on local machine. The latest version of Node.js comes with npm, and it's available on the official Node.js website.

Start by creating an empty directory in your computer. Next, launch the command prompt and cd into the folder you just created:

cd path/to/project

Once pointed to the directory, run the following command to create a Node project:

npm init -y

Next, run the following command to install express and openai libraries from npm:

npm i express openai

create server 

 create a file named server.js inside the root folder of your project. Next, open the file with a code editor add source code
